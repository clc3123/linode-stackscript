linode-stackscript
==================

linode stackscript for bootstrapping

支持：
+   precise64-with-multi-user-rvm-installed

    支持ubuntu 12.04 amd64镜像

    配置选项包含：
    -   ip
    -   hostname
    -   fqdn
    -   time zone
    -   ssh port
    -   deploy user name
    -   deploy user password
    -   deploy user pub key
    -   rvm version
    -   ruby version
